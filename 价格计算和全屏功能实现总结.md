## ✅ 功能1: 商品价格毛利率自动计算

### 需求描述

- 在商品编辑页面支持根据成本价和毛利率自动计算吊牌价
- 根据吊牌价和统一会员折扣自动计算会员价
- 毛利率以百分比显示，支持上下调整
- 统一会员折扣比例在系统配置中维护

### 实现方案

#### 1. 计算公式

**吊牌价计算**:
```javascript
吊牌价 = 成本价 / (1 - 毛利率/100)

示例: 成本价100元，毛利率30%
吊牌价 = 100 / (1 - 0.3) = 142.86元
```

**会员价计算**:
```javascript
会员价 = 吊牌价 × 会员折扣比例

示例: 吊牌价142.86元，会员折扣0.95(95折)
会员价 = 142.86 × 0.95 = 135.72元
```

#### 3. 系统配置实现

**文件**: `frontend/src/pages/system/SystemConfig.tsx`

**新增配置项** (会员设置标签页):
```tsx
<Form.Item
  name="memberDiscountRate"
  label="统一会员折扣比例"
  help="用于计算商品会员价，如0.95表示95折"
  rules={[
    { required: true, message: '请输入会员折扣比例' },
    { type: 'number', min: 0.01, max: 1, message: '折扣比例必须在0.01-1之间' }
  ]}
>
  <InputNumber
    min={0.01}
    max={1}
    step={0.01}
    precision={2}
    formatter={value => `${(Number(value) * 100).toFixed(0)}折`}
    parser={value => {
      const num = value?.replace('折', '') || ''
      return Number(num) / 100
    }}
  />
</Form.Item>
```

**特性**:
- ✅ 显示格式: "95折"
- ✅ 存储格式: 0.95
- ✅ 范围限制: 0.01-1 (1折到100折)
- ✅ 必填验证

### 使用流程

1. **配置会员折扣**:
   ```
   系统管理 → 系统参数配置 → 会员设置
   → 统一会员折扣比例: 输入 0.95 (显示为"95折")
   → 保存配置

2. **添加商品并使用自动计算**:
   ```
   商品管理 → 新增商品
   → 输入成本价: 100元

   - 可以手动修改会员价，不影响吊牌价
   - 调整毛利率会重新计算吊牌价和会员价

## ✅ 功能2: 网页全屏支持

### 需求描述

- 支持网页全屏显示
- 在用户界面提供全屏切换按钮
- 可以进入和退出全屏模式

### 实现方案

**文件**: `frontend/src/pages/layout/MainLayout.tsx`

#### 1. 导入图标
```typescript
import {
  FullscreenOutlined,
  FullscreenExitOutlined
} from '@ant-design/icons'
```

#### 2. 状态管理
```typescript
const [isFullscreen, setIsFullscreen] = useState(false)
```

#### 3. 监听全屏状态
```typescript
useEffect(() => {
  const handleFullscreenChange = () => {
    setIsFullscreen(!!document.fullscreenElement)
  }

  document.addEventListener('fullscreenchange', handleFullscreenChange)
  return () => {
    document.removeEventListener('fullscreenchange', handleFullscreenChange)
  }
}, [])
```

#### 4. 全屏切换函数
```typescript
const toggleFullscreen = async () => {
  try {
    if (!document.fullscreenElement) {
      await document.documentElement.requestFullscreen()
    } else {
      await document.exitFullscreen()
    }
  } catch (err) {
    console.error('全屏切换失败:', err)
  }
}
```

#### 5. UI按钮
```tsx
<Tooltip title={isFullscreen ? '退出全屏' : '进入全屏'}>
  <Button
    type="text"
    icon={isFullscreen ? <FullscreenExitOutlined /> : <FullscreenOutlined />}
    onClick={toggleFullscreen}
    style={{
      fontSize: 18,
      color: mintTheme.primary[600]
    }}
  />
</Tooltip>
```

**位置**: 右上角，在时间显示和用户菜单之间

### 功能特性

- ✅ 动态图标: 全屏时显示"退出全屏"图标，非全屏时显示"进入全屏"图标
- ✅ Tooltip提示: 鼠标悬停显示操作提示
- ✅ 快捷键支持: F11键也可以触发全屏(浏览器原生)
- ✅ 状态同步: 监听fullscreenchange事件，与浏览器全屏状态同步
- ✅ 错误处理: 捕获全屏API调用失败的情况

### 使用方式

1. **点击按钮进入全屏**:
   - 点击右上角全屏图标
   - 整个网页进入全屏模式

2. **退出全屏**:
   - 点击右上角退出全屏图标
   - 按 ESC 键
   - 按 F11 键
