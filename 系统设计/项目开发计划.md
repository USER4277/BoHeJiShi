## 一、项目概述

### 1.1 项目背景

薄荷集市是一家专注于时尚饰品销售的品牌店铺，主营产品包括戒指、项链、手链、耳饰、发饰、胸针等各类时尚配饰。随着业务规模不断扩大，传统的纸质记账和Excel表格管理已无法满足日常运营需求，亟需一套完整的店铺管理系统来提升运营效率、降低管理成本、实现数字化转型。

### 1.2 项目目标

| 目标 | 描述 |
|------|------|
| 业务数字化 | 将商品管理、库存管理、销售管理、财务管理等核心业务环节全部纳入系统管理 |
| 流程自动化 | 通过系统自动化处理日常业务，减少人工操作失误，提高工作效率 |
| 决策数据化 | 提供多维度数据分析和可视化报表，为经营决策提供数据支撑 |
| 服务标准化 | 建立统一的客户服务标准，提升客户满意度和忠诚度 |

### 1.3 项目范围

| 类型 | 内容 |
|------|------|
| 应用平台 | Web端（PC浏览器） |
| 用户规模 | 单管理员用户 |
| 数据规模 | 小规模本地数据 |
| 部署方式 | 本地Mac电脑运行 |
| 网络环境 | 本地局域网访问 |

### 1.4 约束条件

| 约束条件 | 说明 |
|----------|------|
| 运行环境 | 本地Mac电脑，非云服务商 |
| 部署方式 | 直接运行，无需复杂容器化 |
| 数据存储 | SQLite单文件数据库 |
| 用户数量 | 单管理员用户 |
| 网络 | 本地局域网访问 |

## 二、技术架构

### 2.1 技术选型

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **前端框架** | React 18 + TypeScript | 成熟生态，组件化开发，类型安全 |
