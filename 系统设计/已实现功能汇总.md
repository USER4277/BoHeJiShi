### 3. 手工账功能（新增）
**文件位置:**
- Backend: 
  - Schema: `/backend/prisma/schema.prisma` (ManualAccountType, ManualAccount)
  - Routes: `/backend/src/routes/manual-account.ts`
  - Seed: `/backend/scripts/seed-manual-account-types.ts`
- Frontend:
  - API: `/frontend/src/api/manual-account.ts`
  - Page: `/frontend/src/pages/manual-account/ManualAccount.tsx`
  - Route: `/frontend/src/App.tsx:75`
  - Menu: `/frontend/src/pages/layout/MainLayout.tsx:192-203`

**功能说明:**
- 记录额外的收入和支出（非商品销售）
- 收入类型（6种）:
  - 其他营业收入、服务收入、租金收入、利息收入、补贴收入、其他收入
- 支出类型（19种）:
  - 房租、水费、电费、燃气费、物业费、税费、工资、社保公积金
  - 办公用品、装修维修、广告宣传、运输费、通讯费、差旅费
  - 招待费、培训费、保险费、银行手续费、其他支出

**核心功能:**
- 手工账类型管理（CRUD）
- 手工账记录管理（CRUD）
- 支付方式选择（现金、微信、支付宝、银行卡）
- 统计汇总（按类型分类、收支对比）
- 日期范围筛选

**API端点:**
```
GET    /api/manual-account/types         # 获取类型列表
POST   /api/manual-account/types         # 创建类型
PUT    /api/manual-account/types/:id     # 更新类型
DELETE /api/manual-account/types/:id     # 删除类型

GET    /api/manual-account/records       # 获取记录列表（分页）
POST   /api/manual-account/records       # 创建记录
PUT    /api/manual-account/records/:id   # 更新记录
DELETE /api/manual-account/records/:id   # 删除记录

GET    /api/manual-account/statistics    # 获取统计数据
```

### 4. 财务报表整合手工账数据
**文件位置:**
- Backend: `/backend/src/routes/financial.ts`
- Frontend: 
  - `/frontend/src/pages/financial/CashflowReport.tsx`
  - `/frontend/src/pages/financial/ProfitLossReport.tsx`

**功能说明:**
- 现金流量表和损益表支持两种模式:
  1. **完整报表模式**（默认）: 包含经营数据 + 手工账数据
  2. **仅经营模式**: 只显示商品销售相关的经营数据

**现金流量表改进:**
- 经营活动现金流（销售收入、退货支出）
- 额外收入明细（按类型汇总）
- 额外支出明细（按类型汇总）
- 总计包含全部收支

**损益表改进:**
- 营业收入（商品销售）
- 销售成本和毛利
- 额外收入明细
- 额外支出（营业费用）
- 净利润计算包含全部收支
- 同时显示纯经营数据和综合数据对比

**前端特性:**
- 切换开关"仅经营数据"可实时切换显示模式
- 手工账明细以卡片形式分类展示
- 绿色标识收入、红色标识支出

**API变更:**
```
GET /api/financial/cashflow?operatingOnly=true|false
GET /api/financial/profit-loss?operatingOnly=true|false
```

## 数据库变更

### 新增表
```sql
-- 手工账类型
CREATE TABLE ManualAccountType (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  name        TEXT NOT NULL,
  category    TEXT NOT NULL, -- income/expense
  description TEXT,
  sort        INTEGER DEFAULT 0,
  status      INTEGER DEFAULT 1,
  createdAt   DATETIME DEFAULT CURRENT_TIMESTAMP,
  updatedAt   DATETIME,
  UNIQUE(name, category)
);

-- 手工账记录
CREATE TABLE ManualAccount (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  typeId      INTEGER NOT NULL,
  amount      REAL NOT NULL,
  payWay      TEXT,
  date        DATETIME NOT NULL,
  description TEXT,
  operatorId  INTEGER,
  createdAt   DATETIME DEFAULT CURRENT_TIMESTAMP,
  updatedAt   DATETIME,
  FOREIGN KEY (typeId) REFERENCES ManualAccountType(id),
  FOREIGN KEY (operatorId) REFERENCES User(id)
);
```

### User表关系更新
```sql
ALTER TABLE User ADD COLUMN manualAccounts (relation to ManualAccount)
```
