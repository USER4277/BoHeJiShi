## 十、常见问题

### Q1: 端口被占用

**错误**: `Error: listen EADDRINUSE: address already in use :::3001`

**解决**:
```bash
# 查找占用端口的进程
lsof -i :3001
# 或
lsof -i :5173

# 停止进程
kill -9 <PID>
```

---

### Q2: 数据库文件不存在

**解决**:
```bash
cd backend
npx prisma db push
npx tsx src/scripts/init-admin.ts
```

---

### Q3: 权限错误

**解决**:
```bash
chmod +x backend/src/scripts/*.ts
```
