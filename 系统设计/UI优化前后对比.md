## 交互效果对比

### 优化前
```
按钮: 无悬浮效果
卡片: 静态阴影
输入框: 标准蓝色边框
菜单: 标准高亮
```

### 优化后
```
按钮: 
- 悬浮向上移动 2px
- 阴影加深
- 渐变背景加深

卡片:
- 悬浮向上移动 4px
- 阴影从 15% 增加到 25%
- 平滑过渡动画

输入框:
- 焦点时薄荷绿边框
- 外发光效果（rgba(34, 197, 94, 0.1)）

菜单:
- 选中态薄荷绿背景
- 悬浮态浅绿背景
- 平滑过渡动画
```
