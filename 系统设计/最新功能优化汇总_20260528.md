- 格式：成本 ¥XX.XX

## 2. 退货原因结构化管理

### 问题
退货原因之前是随意输入的文本，不利于后期数据分析和AI分析功能扩展。

### 解决方案
提供结构化的退货原因选项，同时支持补充说明。

**退货原因预设选项：**
1. 质量问题
2. 尺码不合适
3. 颜色/款式不喜欢
4. 收到商品与描述不符
5. 商品破损或有瑕疵
6. 买错了/不需要了
7. 价格原因
8. 服务态度问题
9. 顾客个人原因
10. 其他原因

**功能特性：**
- 下拉选择框，可搜索
- 必填字段（验证）
- 可选的备注说明框（最多200字）
- 最终保存格式：`原因 - 备注`（如有备注）

### 数据结构
```javascript
// 订单返回详情中包含
{
  returns: [
    {
      id: number,
      returnNo: string,
      returnReason: string,  // 结构化原因
      returnAmount: number,
      createdAt: datetime
    }
  ]
}
```

### 退货信息展示
订单详情页面新增退货信息卡片：
- 红色主题背景 (#fff1f0)
- 显示：退货单号、退款金额、退货原因、退货时间
- 支持多次退货记录（如有多次部分退货）

- `/backend/src/routes/sale.ts`
  - 第 178-188 行：订单详情 API 包含 returns 关联

### 为AI分析做准备
结构化数据便于后续分析：
- 统计各类退货原因占比
- 识别商品质量问题
- 分析服务改进点
- 预测退货风险

## 3. UI布局优化 - 修复导航栏与主内容重叠

### 问题
左侧导航栏与右侧主页面内容有重叠现象。

### 解决方案

**左侧导航栏（Sider）:**
- 固定定位 (`position: fixed`)
- 明确宽度：展开 250px，收起 80px
- 固定在左侧 (`left: 0, top: 0, bottom: 0`)
- 高层级 (`z-index: 100`)
- 可滚动 (`overflow: auto`)

**右侧主内容区（Layout）:**
- 动态左边距：
  - 导航展开时：`marginLeft: 250px`
  - 导航收起时：`marginLeft: 80px`
- 平滑过渡：`transition: margin-left 0.2s`

**顶部导航栏（Header）:**
- 粘性定位 (`position: sticky, top: 0`)
- 始终可见 (`z-index: 99`)
- 跟随主内容区宽度自适应

### 视觉效果
- ✅ 无重叠
- ✅ 导航切换时内容区平滑过渡
- ✅ 滚动时头部固定
- ✅ 响应式布局

### 修改文件
- `/frontend/src/pages/layout/MainLayout.tsx`
  - 第 243-256 行：Sider 固定定位和尺寸
  - 第 303-315 行：主内容区动态边距
  - 第 305-317 行：Header 粘性定位

## 技术细节

### 成本价显示
```tsx
{
  title: '成本价',
  dataIndex: 'costPrice',
  key: 'costPrice',
  align: 'right' as const,
  render: (val: number) => val ? (
    <span style={{ color: '#f97316', fontWeight: 500 }}>
      ¥{val.toFixed(2)}
    </span>
  ) : '-'
}
```

### 退货原因表单

```tsx
<Form.Item

```

```tsx
  width={250}

    position: 'fixed',
```
