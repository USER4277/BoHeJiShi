# 退货积分处理测试场景

## 场景说明

验证退货时正确处理积分抵扣和积分返还

## 测试用例 1: 使用积分抵扣的订单退货

### 初始状态
- 会员积分: 1000 分
- 会员消费总额: 0 元

### 下单
- 商品总价: 200 元
- 使用积分: 500 分抵扣 (500 × 0.002 = 1 元)
- 实际支付: 199 元
- 获得积分: 199 分 (按实际支付金额)

**订单数据:**
```javascript
{
  totalAmount: 200,
  discountAmount: 1,      // 积分抵扣
  payAmount: 199,         // 实际支付
  pointsUsed: 500,        // 使用的积分
  pointsEarned: 199       // 获得的积分
}
```

**下单后会员状态:**
- 积分: 1000 - 500 + 199 = 699 分
- 消费总额: 199 元

### 退货

**应该发生的操作:**

1. **退款金额**: 199 元 (实际支付的现金)
   - ✓ 不包含积分抵扣的 1 元

2. **积分处理:**
   - 返还使用的积分: +500 分
   - 扣除获得的积分: -199 分
   - 净变化: +301 分

3. **消费总额**: -199 元

**退货后会员状态:**
- 积分: 699 + 301 = 1000 分 (恢复到下单前)
- 消费总额: 199 - 199 = 0 元 (恢复到下单前)

**积分日志:**
```javascript
[
  // 下单时
  { type: 'exchange', points: -500, description: '消费抵扣 500 积分' },
  { type: 'earn', points: +199, description: '消费获得积分' },
  
  // 退货时
  { type: 'refund', points: +500, description: '退货返还消费抵扣积分' },
  { type: 'refund', points: -199, description: '退货扣除消费获得积分' }
]
```

## 关键规则总结

1. **退款金额 = 实际支付金额 (payAmount)**
   - 不包含积分抵扣的部分
   - 积分抵扣部分通过返还积分来处理

2. **退货时积分处理:**
   - 返还: 订单使用的积分 (pointsUsed)
   - 扣除: 订单获得的积分 (pointsEarned)
   - 净变化 = +pointsUsed - pointsEarned

3. **消费总额调整:**
   - 减少实际支付金额 (payAmount)
   - 与退款金额一致

4. **库存恢复:**
   - 所有订单商品数量恢复到库存

## 数据库变更

### Order 表新增字段
```sql
ALTER TABLE Order ADD COLUMN pointsUsed INTEGER DEFAULT 0;
```

**字段说明:**
- `pointsUsed`: 订单使用的积分数量（用于抵扣现金）
- `pointsEarned`: 订单获得的积分数量（消费奖励）

## 代码实现要点

### 创建订单时
```javascript
const pointsDeduction = usePoints * 0.002;
const payAmount = Math.max(0, totalAmount - discountAmount - pointsDeduction);
const pointsEarned = Math.floor(payAmount); // 按实际支付金额计算

await prisma.order.create({
  data: {
    // ...
    pointsUsed: usePoints,    // ✓ 保存使用的积分
    pointsEarned,             // ✓ 保存获得的积分
    payAmount                 // ✓ 实际支付金额
  }
});
```

### 退货时
```javascript
// 退款金额 = 实际支付金额
const returnAmount = order.payAmount;

let pointsChange = 0;

// 返还使用的积分
if (order.pointsUsed > 0) {
  pointsChange += order.pointsUsed;
}

// 扣减获得的积分
if (order.pointsEarned > 0) {
  pointsChange -= order.pointsEarned;
}

// 更新会员
await prisma.member.update({
  where: { id: order.memberId },
  data: {
    points: { increment: pointsChange },
    totalConsume: { decrement: order.payAmount }
  }
});
```

```sql
  orderNo,
  totalAmount,
  discountAmount,
  payAmount,
  pointsUsed,
  pointsEarned,
  status
FROM Order 
WHERE orderNo = '订单号';

-- 查看会员积分变化
SELECT 
  changeType,
  points,
  balance,
  description,
  createdAt
FROM PointsLog 
WHERE memberId = 会员ID 
  AND orderId = 订单ID
ORDER BY createdAt;

-- 验证会员最终状态
SELECT 
  points,
  totalConsume
FROM Member 
WHERE id = 会员ID;
```
